# Structura - Landing Page

<h2>Description:</h2>
Structura is a single-page HTML5 website. This website is appropriate for websites that focus on construction. The Bootstrap framework was used to build Structura, which has five sections: About, Services, Projects, Testimonials, and Contact. It has a testimonials section, a functioning contact form, and more. The layout is completely responsive.

<h2>Languages and Utilities Used</h2>

- <b>HTML</b>
- <b>CSS</b>
- <b>Bootstrap</b>
- <b>Figma</b>
